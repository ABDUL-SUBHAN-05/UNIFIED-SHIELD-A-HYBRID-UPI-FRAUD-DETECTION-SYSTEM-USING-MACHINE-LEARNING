# UNIFIED-SHIELD-A-HYBRID-UPI-FRAUD-DETECTION-SYSTEM-USING-MACHINE-LEARNING<br>
 The project UnifiedShield: A Hybrid ML-Based UPI Fraud Detection System demonstrates how machine learning can enhance security in digital payment platforms like UPI. By analyzing behavioral and transactional patterns, the system improves fraud detection beyond traditional rule-based methods. Lightweight ML models such as Decision Trees and Logistic Regression help classify transactions accurately while maintaining simplicity and efficiency. The inclusion of a Streamlit dashboard enables clear visualization of alerts and model outputs, making the system easy to use and understand. Overall, this prototype highlights the potential of ML-driven fraud detection and serves as a foundation for scalable, real-time, and context-aware UPI security solutions.
